# Групповой проект 2 по СМАДИМО

## API и парсинг

_Контекст кейса и бизнес-задача:_

Краудфандинговая платформа “Grass” (зарабатывает на комиссии за размещение проекта, а также на процентах от общего сбора проекта) захотела увеличить выручку за счёт внедрения нового функционала, позволяющего определять вероятную успешность нового проекта, а также на какие сборы может рассчитывать автор. 
Основная задача проекта: Проанализировать данные с аналогичных платформ и написать алгоритм, который, опираясь на категорию, тему, описание, требуемую сумму финансирования, будет выдавать автору проекта вероятность того, что его проект станет успешным (соберет нужную сумму) и укажет, в случае низкой вероятности успеха, на какие сборы,  может рассчитывать автор. 
Обоснование задачи: Так как заказчик захотел внедрить функционал, который должен предсказывать, перед нами встала задача обучения модели и для этого нам понадобится большой объем данных о проектах и их сборах, который мы будем собирать с других краудфандинговых платформ. Основная информация о проекте содержится в следующих признаках: категория, тема, описание, а также требуемая сумма финансирования, поэтому именно эти данные мы выбрали для анализа и дальнейшего обучения.
